Project 1: How Got Em

Si tratta del progetto full-stack che ho realizzato come capstone project di Epicode. Per il frontend ho utilizzato Angular, con Angular Material come framework CSS, mentre per il backend ho utilizzato Java con Spring Boot. Il database utilizzato è PostgreSQL. Si tratta di un ecommerce di sneakers con autenticazione basata su Token JWT, con ruoli admin e user. Inoltre, ho utilizzato l'API di Google per il reCaptcha e l'API di PayPal per il pagamento, testando il tutto con account sandbox. Al momento dell'acquisto, l'ordine e i dati PayPal vengono salvati nel database e viene inviata una mail ad entrambe le parti, grazie a Spring Boot Mail.

Se sei interessato, puoi trovare il codice del progetto nella repository GitHub, dove troverai anche le istruzioni per avviare l'applicazione nel file README.md.
